# Calculadora de Bhaskara em Java

## 📌 Descrição
Este projeto foi desenvolvido em Java para calcular as raízes de uma equação do 2º grau utilizando a fórmula de Bhaskara.

## 🧮 Fórmula utilizada

Δ = b² - 4ac

x1 = (-b + √Δ) / 2a  
x2 = (-b - √Δ) / 2a  

## 🚀 Funcionalidades
- Entrada dos valores A, B e C
- Cálculo do Delta
- Cálculo das raízes
- Validação para Delta negativo

## 🛠️ Tecnologias utilizadas
- Java
- Scanner
- Programação Orientada a Objetos

## ▶️ Como executar
1. Executar o arquivo Main.java
2. Inserir valores de A, B e C
